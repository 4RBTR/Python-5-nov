# Analisis Data Nilai Mapel

1. Mapel mana yang memiliki rata-rata nilai tertinggi?
   jawaban: Jadi, mapel dengan rata-rata nilai tertinggi adalah Fisika (rata-rata = 89,33).
2. Mapel mana yang memiliki nilai terendah?
   jawaban: Nilai terendah secara keseluruhan berasal dari Bahasa Indonesia (60).
3. Bagaimana visualisasi membantu dalam memahami data?
